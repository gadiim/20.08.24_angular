Тема: Створення простих компонентів на Angular

 Завдання 1: Компонент для відображення подій

1. Створіть компонент `event-item`:
   - Створіть компонент, який відображає інформацію про подію, таку як назва події, 
   дата та опис. Додайте властивості до компонента для передачі даних про подію.

2. Оновіть шаблон компонента:
   - Внесіть зміни в шаблон компонента, щоб відобразити передані дані про подію, 
   включаючи назву, дату та опис.

  Завдання 2: Використання компонента з параметрами

1. Включіть компонент `event-item` у шаблон батьківського компонента:
   - Вставте компонент `event-item` у шаблон батьківського компонента 
   (наприклад, `app.component`). Передайте дані про подію через властивості 
   компонента. Дані повинні включати назву події, дату та опис.

2. Перевірте результат:
   - Переконайтеся, що компонент `event-item` коректно відображає дані про подію, 
   які були передані через властивості.

  Завдання 3: Додання кількох подій

1. Створіть список подій у батьківському компоненті:
   - У батьківському компоненті створіть масив подій, де кожна подія має назву, 
   дату та опис.

2. Використовуйте цикл для відображення компонентів `event-item`:
   - Використовуйте цикл для відображення компонента `event-item` для кожної 
   події у списку. Переконайтеся, що кожен компонент відображає правильну 
   інформацію про подію.
